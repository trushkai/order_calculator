# Order Calculator MVP

## Стек
- Frontend: React, TypeScript, Ant Design
- Backend: Node.js, Express
- Database: PostgreSQL, Prisma

## Запуск
1. npm install
2. заполнить apps/server/.env
3. npm run prisma:migrate -w apps/server
4. npm run server:dev
5. npm run client:dev

## Роли
- Администратор
- Менеджер

## Возможности
- CRUD справочников
- Создание шаблонов калькуляторов
- Динамический рендер формы
- Реактивный пересчет итога
