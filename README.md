# Order Calculator MVP

Веб-приложение для создания и использования шаблонов расчёта стоимости заказов.

Проект реализует два режима работы:
- **Администратор** — создает справочники, шаблоны, поля и формулы.
- **Менеджер** — выбирает готовый шаблон, вводит значения и мгновенно получает итоговую сумму.

## Возможности

### Администратор
- создание и редактирование справочников;
- добавление значений справочников;
- создание и редактирование шаблонов калькуляторов;
- добавление динамических полей:
  - число;
  - выпадающий список;
- привязка поля типа `select` к существующему справочнику;
- составление формул через ключи полей, например:

```text
#{weight} * #{count} + #{tariff}
```

- удаление шаблонов;
- удаление справочников, если они не используются в шаблонах.

### Менеджер
- выбор калькулятора из списка;
- динамический рендер формы по выбранному шаблону;
- автоматический пересчет результата без кнопки «Рассчитать»;
- расчет формул на лету через `mathjs`.

## Архитектура базы данных

Основные сущности:
- `Template` — шаблон калькулятора;
- `TemplateField` — поля шаблона;
- `Dictionary` — справочник;
- `DictionaryValue` — значения справочника;
- `Formula` — формула расчета.

## Требования для запуска

Перед запуском должны быть установлены:
- Node.js

## Настройка окружения

Создайте файл `apps/server/.env` по примеру `apps/server/.env.example`.

Пример:

```env
PORT=4000
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/order_calculator?schema=public"
CLIENT_URL="http://localhost:5173"
```

## Установка зависимостей

В корне проекта выполните:

```bash
npm install
```

## Запуск проекта

### 1. Применить миграции Prisma

```bash
npm run prisma:migrate -w apps/server
```

### 2. Запустить backend

```bash
npm run server:dev
```

### 3. Запустить frontend

В отдельном терминале:

```bash
npm run client:dev
```

## Пример сценария использования

### 1. Создание справочника
Создать справочник, например:
- `Тариф доставки`
- значения:
  - Эконом — `100`
  - Стандарт — `200`
  - Срочно — `350`

### 2. Создание шаблона
Создать шаблон `Расчет доставки` с полями:
- `Вес груза` → `weight`
- `Количество мест` → `count`
- `Тариф` → `tariff`

Формула:

```text
#{weight} * #{count} + #{tariff}
```

### 3. Проверка в режиме менеджера
Ввести:
- вес = `10`
- количество = `3`
- тариф = `200`

Результат:

```text
230
```
