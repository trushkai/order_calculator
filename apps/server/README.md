# Server

## Запуск
```bash
npm run dev
```

## Основные маршруты
- `GET /api/health`
- `GET /api/dictionaries`
- `GET /api/dictionaries/:id`
- `POST /api/dictionaries`
- `PUT /api/dictionaries/:id`
- `GET /api/templates`
- `GET /api/templates/:id`
- `POST /api/templates`
- `PUT /api/templates/:id`
